Essa interface, do Power Automate para desktop, mostra uma de minhas automações que interage com a interface do usuário para efetuar o lançamento manual e registro de notas fiscais em um sistema conectado a um banco de dados. O processo gerou significativo ganho de tempo para realização de outras atividades que exigem intervenção direta de pessoas.
O código em python mostrado ao lado é um exemplo de vários outros códigos criados por mim e utilizados para automações web e tratamento de dados no geral. Nesse cenário a biblioteca Selenium foi utilizada para interação com elementos em javascript, além disso, utilizo a biblioteca pandas para criação de arquivos em .csv e análise/tratamento de dados.
Atualmente estou realizando trabalhos de pesquisas com o Neo4j, um banco de dados orientado a grafos, muito popular na área de negócios. A estruturação e filtragem de informações importantes para tomada de decisão é um ponto forte desse DataBase, alguns de meus estudos envolvem a extração de dados de uma consulta API que retorna informações de tweets do antigo Twitter, em formato JSON.
